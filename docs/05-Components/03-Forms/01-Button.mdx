# Button

Button component is used to trigger an action or event, such as submitting a form, opening a Dialog, canceling an action, or performing a delete operation.

## Import 

```shell
const { Button } from '@margaret-ui/core';
```

## Usage 

```jsx live
<Stack gap={1} alignY="center">
  <Button variant="solid">Hello Margaret!</Button>
 
</Stack>
```

### Button Variants

Use the `variant` prop allows to change the styled of the Button. 

```jsx live
<Stack gap={1} alignY="center">
  <Button variant="solid">Hello Margaret!</Button>
  <Button variant="outline">Hello Margaret!</Button>
  <Button variant="ghost">Hello Margaret!</Button>
  <Button variant="link">Hello Margaret!</Button>
</Stack>
```

### Button Sizes

Four sizes are provided by default. Use the `size` prop allows to change the padding, the font-size and the min-width. They are customisable in the theme. 

```jsx live
<Stack gap={1} alignY="center">
  <Button size="large">Hello Margaret!</Button>
  <Button size="default">Hello Margaret!</Button>
  <Button size="small">Hello Margaret!</Button>
  <Button variant="link" size="bare">Hello Margaret!</Button>
</Stack>
```

### Button with Icon

It's possible to pass a React element to the `rightIcon` or the `leftIcon` prop. Use the `gap` prop to change the spacing bectween icons and the button text. 

```jsx live
<Stack gap={1} alignY="center">
  <Button variant="solid" leftIcon={<IoBulbOutline size={20} />} gap={0.5}>Hello Margaret!</Button>
  <Button variant="outline" rightIcon={<IoChevronForwardOutline size={16} />} gap={0.5}>See more</Button>
</Stack>
```

## Theme

### Variant Theme
For each variant, it's possible to customizable this property : 

```javascript
  fontWeight: string | number,
  fontSize: string,
  borderRadius: string,
  alignY: string,
  alignX: string,
  transition: string,

  background: string,
  color: string,
  boxShadow: string,
  textDecoration: string,

  backgroundHover: string,
  colorHover: string,
  boxShadowHover: string,
  textDecorationHover: string,


  backgroundFocus: string,
  colorFocus: string,
  boxShadowFocus: string,
  textDecorationFocus: string,

  backgroundDisabled: string,
  colorDisabled: string,
  boxShadowDisabled: string,
  textDecorationDisabled: string,
  opacityDisabled: number,
```

### Size Theme

For each size, it's possible to customizable this property : 

```javascript
  paddingVertical: string | number,
  paddingHorizontal: string | number,
  minWidth: string,
  fontSize: string,
```
