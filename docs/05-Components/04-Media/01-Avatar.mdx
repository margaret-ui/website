# Avatar

The Avatar component is used to represent a user, and displays the profile picture, initials or fallback icon.

## Import 

```js
import { Avatar } from '@margaret-ui/core';
```

## Usage 

```jsx live
<Stack gap={1}>
  <Avatar
    name="Justin Case"
  />
  <Avatar
    imageUrl="https://bit.ly/3J7fv4q"
    name="Sue Shei"
  />
  <Avatar
    imageUrl="https://bit.ly/3Ozl5Oe"
    name="Hugh Saturation"
  />
  <Avatar
    imageUrl="https://bit.ly/3RXkn09"
    name="Rodney Artichoke"
  />
  <Avatar>P.T</Avatar>
  <Avatar
    imageUrl="https://bit.ly/3RVNSPT"
    name="Max Conversion"
  />
</Stack>
```

### Avatar Sizes

The Avatar component is available in 6 sizes by default.
- `tiny` - 1rem
- `small` - 2rem
- `default` - 3rem
- `medium` - 4rem
- `large` - 6rem
- `huge` - 8rem

```jsx live
<Stack gap={1}>
  <Avatar
    name="Justin Case"
    size="tiny"
  />
  <Avatar
    imageUrl="https://bit.ly/3J7fv4q"
    name="Sue Shei"
    size="small"
  />
  <Avatar
    imageUrl="https://bit.ly/3Ozl5Oe"
    name="Hugh Saturation"
    size="default"
  />
  <Avatar
    imageUrl="https://bit.ly/3RXkn09"
    name="Rodney Artichoke"
    size="medium"
  />
  <Avatar size="large">P.T</Avatar>
  <Avatar
    imageUrl="https://bit.ly/3RVNSPT"
    name="Max Conversion"
    size="huge"
  />
</Stack>
```

### Avatar Shapes

Three shapes are provided by default. They set the avatar as a circle (`default`), `rounded` or `square`.

```jsx live
<Stack gap={1}>
  <Avatar
    name="Justin Case"
    shape="default"
    size="medium"
  />
  <Avatar
    imageUrl="https://bit.ly/3J7fv4q"
    name="Sue Shei"
    shape="rounded"
    size="medium"
  />
  <Avatar
    imageUrl="https://bit.ly/3Ozl5Oe"
    name="Hugh Saturation"
    shape="square"
    size="medium"
  />
</Stack>
```

### Customize Avatar

You can customize the background color, the font size, the icon placeholder and show a border to match with your design.
- To update the background, pass a color in the `bg` prop
- To update the font size, pass a number in the `fontSize` prop which match with the number of rem of your size
- To update the placeholder, pass the `placeholder` property
- To show a border, pass the `showBorder` prop

```jsx live
<Stack gap={1}>
  <Avatar
    name="Justin Case"
    size="medium"
    bg="rebeccapurple"
  />
  <Avatar
    name="Sue Shei"
    size="medium"
    fontSize={1}
  />
  <Avatar
    size="medium"
    placeholder={<IoBulbOutline size={38} />} 
  />
  <Avatar
    name="Rodney Artichoke"
    size="medium"
    showBorder
  />
</Stack>
```

### Avatar Fallback

The Avatar component displays the item in the following order of importance: 

1. `imageUrl`
2. If we don't have an imageUrl, we use the `children`
2. `name`, we use this prop to generate the initials
3. `placeholder`, you can customize the placeholder to be displayed if no other property is known


```jsx live
<Stack alignY="center" gap={1}>
  <Avatar
    imageUrl="https://bit.ly/3Ozl5Oe"
    name="Hugh Saturation"
    size="medium"
    placeholder={<IoBulbOutline size={38} />} 
  >
    NM
  </Avatar>
  <Avatar
    name="Hugh Saturation"
    size="medium"
    placeholder={<IoBulbOutline size={38} />} 
  >
    NM
  </Avatar>
  <Avatar
    name="Hugh Saturation"
    size="medium"
    placeholder={<IoBulbOutline size={38} />} 
  />
  <Avatar
    size="medium"
    placeholder={<IoBulbOutline size={38} />} 
  />
  <Avatar
    size="medium"
  />
</Stack>
```

### Avatar Group

In some cases, you might need to stack avatars as a group. Use the Avatar.Group component.

- To limit the number of avatars to show, use the `max` prop
- To size all the avatars equally, pass the `size` prop 
- To set the shape all the avatars equally, pass the `shape` prop 
- To adjust the spacing between the avatars, pass the `spacing` prop
- To display a border on each avatar, pass the `showBorder` prop

A `zIndex` prop is available for the Avatar component. You can adjust it if you want to play with the layout of the avatars in the avatar group.


```jsx live
<Avatar.Group showBorder size="medium" spacing={-1.5}>
  <Avatar
    name="Justin Case"
    bg="tomato"
  />
  <Avatar
    imageUrl="https://bit.ly/3J7fv4q"
    name="Sue Shei"
  />
  <Avatar
    imageUrl="https://bit.ly/3Ozl5Oe"
    name="Hugh Saturation"
  />
  <Avatar
    imageUrl="https://bit.ly/3RXkn09"
    name="Rodney Artichoke"
  />
  <Avatar>P.T</Avatar>
  <Avatar
    imageUrl="https://bit.ly/3RVNSPT"
    name="Max Conversion"
  />
</Avatar.Group>
```

### getInitials function

If you need to calculate the initial of a name outside the Avatar component, you can use the function Avatar.getInitials

```jsx live
<Text>
  {Avatar.getInitials("Gordon Norman")}
</Text>
```

## Theming

You can easily customise your avatars by adding elements to your theme. 

Here's the default theme for the avatar. You can update all elements or add a new shape or a new size if you need it.

```javascript
{
  ...theme, 
  avatar: {
    background: 'var(--ui-primary)',
    backgroundLight: 'var(--ui-primary-light)',
    border: '2px solid #fff',
    color: '#fff',

    shape: {
      default: '100%',
      rounded: '6px',
      square: '0',
    },

    sizes: {
      tiny: '1rem',
      small: '2rem',
      default: '3rem',
      medium: '4rem',
      large: '6rem',
      huge: '8rem',
    },
  }
}
```
