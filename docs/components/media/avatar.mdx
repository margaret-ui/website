The Avatar component is used to represent a user, and displays the profile picture, initials or fallback icon.

## Import

```shell
const { Avatar } from '@margaret-ui/cogit status re';
```

## Usage

```jsx live
<Stack alignY="center" gap={1}>
  <Avatar
    firstName="Cynthia"
    lastName="Henaff"
    imageUrl={'https://cutt.ly/Vn4U6A0'}
  />
  <Avatar
    firstName="Henri"
    lastName="Michel"
    imageUrl={'https://cutt.ly/2n4Iy3n'}
  />
  <Avatar firstName="Sonia" lastName="Si Moussi" />
  <Avatar name="Beno√Æt Carquillat" />
</Stack>
```

### Sizes

The Avatar component is available in 6 sizes in the default theme.

```js title="theme.js"
const theme = {
  ...
  avatar : {
    sizes: {
      tiny: '1rem',
      small: '2rem',
      default: '3rem',
      medium: '4rem',
      large: '6rem',
      huge: '8rem',
    },
  }
};
```

```jsx live
<Stack alignY="center" gap={1}>
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" size="tiny" />
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" size="small" />
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" />
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" size="medium" />
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" size="large" />
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" size="huge" />
</Stack>
```

It's possible to add sizes or modify existing ones by defining a custom `theme.avatar.sizes`. For instance, setting `theme.avatar.size` as follow‚Ä¶

```js title="theme.js"
const theme = {
  ...
  avatar : {
    sizes: {
      micro: '32px',
      default: '96px',
      mega: '256px',
    }
  },
};
```

‚Ä¶would give this results in-app.

```jsx live
<ThemeProvider
  theme={{
    avatar: {
      sizes: {
        micro: '32px',
        default: '96px',
        mega: '256px',
      },
    },
  }}
>
  <Stack alignY="center" gap={1}>
    <Avatar imageUrl="https://tym8.link/baseball-sort-push" size="micro" />
    <Avatar imageUrl="https://tym8.link/baseball-sort-push" />
    <Avatar imageUrl="https://tym8.link/baseball-sort-push" size="mega" />
  </Stack>
</ThemeProvider>
```

### Shape

Three shapes are provided in the defaukt theme. They set the avatar as a circle (`default`), `rounded` or `square`.

```jsx live
<Stack alignY="center" gap={1}>
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" />
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" variant="rounded" />
  <Avatar imageUrl="https://tym8.link/baseball-sort-push" variant="square" />
</Stack>
```

### Avatar Fallback & Placeholder

You can customize the icon of the fallback avatar icon to match your design requirements.

```jsx live
<Stack alignY="center" gap={1}>
  <Avatar icon={<IoBulbOutline size={38} />} size="medium" />
</Stack>
```

## Theming

The background and border radius can be overriden at the component level.
If set, they take the value of the corresponding theme prop.

For instance, using this theme‚Ä¶

```js title="theme.js"
const theme = {
  ...
  avatar: {
    background: "linear-gradient(to bottom, #f5aba5, #406fb3)",
    color: "#fff",
    border: "",
    shape: {
      default: '100%',
      rounded: '6px',
      square: '0',
    },
    sizes: {
      tiny: '1rem',
      small: '2rem',
      default: '3rem',
      medium: '4rem',
      large: '6rem',
      huge: '8rem',
    }
  }
};
```
